Protocole de Circuit Inter-Intégré

Le Bus I²C

Un protocole de communication série permettant aux microcontrôleurs de dialoguer avec des dizaines de périphériques via seulement deux fils — une idée révolutionnaire née chez Philips Semiconductors en 1982.

Version du protocole 7.0 · Philips / NXP

Protocole

Comment ça Marche

01
CONDITION DE DÉPART (START)

Le maître tire SDA à l'état BAS pendant que SCL est à l'état HAUT. Ce motif unique signale à tous les appareils que la transmission commence.

02
ADRESSE + BIT R/W

Le maître envoie une adresse de 7 bits suivie d'un bit Lecture/Écriture. Chaque esclave écoute et compare avec sa propre adresse.

03
ACQUITTEMENT (ACK)

L'esclave adressé tire SDA à BAS pendant un cycle d'horloge pour acquitter. Si aucun appareil ne répond, le maître détecte un NACK.

04
TRANSFERT DE DONNÉES + STOP

Les octets de données circulent MSB en premier, chacun suivi d'un ACK. Une condition STOP (SDA HAUT pendant SCL HAUT) clôt la transaction.

Monde Réel

Où Vit l'I²C

🌡️
Capteurs
Capteurs de température, humidité, pression et mouvement comme le BME280, MPU-6050 et AHT20.
🖥️
Écrans OLED
Écrans OLED 128×64 basés sur le SSD1306, très répandus dans les projets embarqués.
🔋
Gestion de Batterie
Des CI comme le BQ27441 fournissent la jauge de carburant et la gestion de l'énergie via I²C.
🎛️
Expandeurs GPIO
Le MCP23017 ajoute 16 broches E/S supplémentaires à un microcontrôleur via deux simples fils.
💾
Mémoire EEPROM
Les EEPROM de la série AT24C stockent la configuration et les données persistantes via I²C.
🕐
Horloges Temps Réel
Les RTC DS3231 et PCF8563 maintiennent une heure précise avec une simple connexion I²C.
Implémentation

Exemple Arduino

Lecture de la température depuis un capteur AHT20 avec la bibliothèque Wire sur Arduino — une transaction I²C typique en pratique.

C++ · Arduino 
// Lecture Température I²C — Capteur AHT20
#include <Wire.h>

#define AHT20_ADDR  0x38  // Adresse I²C 7 bits
#define CMD_TRIGGER 0xAC  // Déclencher la mesure

void setup() {
  Wire.begin();          // Initialiser I²C en mode maître
  Serial.begin(115200);
}

float lireTemperature() {
  // Envoyer la commande de déclenchement à l'AHT20
  Wire.beginTransmission(AHT20_ADDR);
  Wire.write(CMD_TRIGGER);
  Wire.write(0x33); Wire.write(0x00);
  Wire.endTransmission();

  delay(80);  // Attendre la fin de la mesure

  // Demander 6 octets au capteur
  Wire.requestFrom(AHT20_ADDR, (uint8_t)6);
  uint8_t data[6];
  for (int i = 0; i < 6; i++) data[i] = Wire.read();

  // Extraire la température brute sur 20 bits
  uint32_t tempBrut = (((uint32_t)data[3] & 0x0F) << 16)
                    | ((uint32_t)data[4] << 8)
                    | data[5];

  return ((float)tempBrut / 1048576.0f) * 200.0f - 50.0f;
}

void loop() {
  float temp = lireTemperature();
  Serial.print("Température : ");
  Serial.print(temp, 2);
  Serial.println(" °C");
  delay(2000);
}
